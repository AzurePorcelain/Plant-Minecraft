【免责声明】
- 自行修改整合包（包括增删或删改模组版本）将不再提供任何技术支持（包括问题解答）。
- 部分模组为整合特供版本，自行更换可能导致游戏出错或部分内容无法正常游玩。

【游玩前必读】
1. 必须使用 64 位 Java 11.0.12，否则可能导致性能问题或崩溃。若使用 Java 8，请先删除 ModernUI 模组。
2. 新建世界时请勿改动默认设置，否则出现问题团队不负责。
3. 禁止同时开启多个客户端，否则可能导致配置文件损坏，需完全重装整合包。
4. 联机时请确保所有玩家处于同一难度，否则将影响非主机玩家体验。
5. UUID 变化会导致玩家数据丢失，详见频道常见问题。

【关于 0.9.0 正式版】
- 附带 StayTrue 材质包，可替代 BetterFoliage 部分功能，默认关闭，可手动开启。
- 未提供官方服务端，因包含服务器不兼容模组 Spiritcaller Backport。如有需要可自行制作服务端并移除该生物，但可能影响部分剧情体验。团队暂无制作服务端计划。

【难度切换说明】
- 切换难度前请完全关闭游戏，否则可能导致存档损坏。
- 建议在开档前确定难度。联机时所有玩家难度需一致。
- 默认难度为标准。切换难度时输入“cheat”可开启测试模式（可使用作弊指令）。
- 注意：测试模式一旦开启无法关闭，仅对新存档生效（或删除旧存档中“serverconfig”文件夹）。
- 多人游戏可调整 config\pvz-common 中的 PlayerCount 选项以启用经验补偿（新版中可单人种植植物独享经验）。
- 新增实验性选项“严苛性别墅生成”：开启后戴夫别墅仅生成于平地，减少地形改造工作，但数量会减少。默认关闭，可在 config\pvz-common 中开启。
- 非 Windows 系统请参考 BAT 脚本手动切换难度。


【难度介绍】（以下为图表未列出的补充说明）

- 故事/休闲
  * 更多的基础资源
  * 无需建筑方块工事也可轻松通关

- 标准
  * 较多的基础资源
  * 你可以选择改善植物阵容或搭建自己的MC方块防御工事

- 残酷（原石粒）
  * 更容易出现珍稀宝藏
  * 可附魔灵魂绑定来防止装备掉落
  * 随着玩家增强的关卡僵尸
  * 可能需要良好的阵容搭配意识或一定的工事搭建能力
  * 适合寻求挑战的玩家，新手请谨慎选择

- 极限
  * 在残酷难度基础上增加"死亡销毁"机制
  * 死亡将销毁除主副手和快捷栏以外的所有背包物品、盔甲、饰品及全部经验
  * 可附魔灵魂绑定来防止装备直接掉落，但仍会消耗大量耐久
  * 特别注意：即使快捷栏物品不掉落也会扣除60%耐久
  * 附魔灵魂绑定可将耐久损耗降低至20%，且耐久附魔对此次扣除有效
  * 强烈不推荐一周目游玩

- 绝境（原噩梦）
  * 更容易出现珍稀宝藏
  * 可附魔灵魂绑定来防止装备掉落
  * 随着玩家增强的关卡僵尸
  * 可能需要非常强的阵容搭配意识或工事搭建能力
  * 不推荐一周目游玩

- 对抗模式
  * 可选休闲、标准、残酷三种难度
  * 死亡保留盔甲，掉落非主副手和快捷栏以外的背包物品、饰品及大量经验
  * 尸体一段时间后会变为骨架，所有玩家均可拾取
  * 具体对抗规则由玩家自行约定（如相互攻击、竞速等）
  * 本模式自由度较高，将选择权交给玩家

【难度通用提示】
1. 不同难度在植物数量上限、关卡难度、BOSS强度及战利品表上有显著差异，请根据自身实力选择。
2. “绝境”是功能性塔防，更关注每个植物个体的极限战力。玩家对关键战力的安排决策将对战局起到决定性作用。
3. 如需中途更改难度，请参阅频道常见问题，删去游戏存档内“serverconfig”文件夹。
